UCSD Triple C

Position: Full Stack Developer

Time:Aug 2021 – March 2023

I collaborated on the Lable Web project, an innovative platform that empowers individual artists to showcase, trade, and communicate through various mediums, including artworks, NFTs, and a dynamic blog system. In my role, I led the frontend development using technologies such as React.js, CSS, HTML, JavaScript, and leveraged the Bootstrap grid and Flexbox for ensuring responsiveness across different devices. Additionally, I took charge of designing and developing the backend systems, which were built using Python, Django, SQL databases, and web sockets. These backend systems were crucial in providing seamless functionality and efficient data management, contributing to the overall success of the project
