Smart Grocery Shopping App

Project Description

This mobile app transforms the grocery shopping experience by integrating automation, AI, and thoughtful UX design to address common frustrations such as overspending, disorganization, and food waste. The app streamlines budgeting, meal planning, pantry management, and store navigation into one cohesive system.

Designed with Human-Computer Interaction (HCI) principles in mind, the app reduces user effort and decision fatigue. The home screen uses preview widgets to display essential information like spending progress, pantry alerts, and meal plans, reducing cognitive load and ensuring smooth navigation.

Key Features

  • Receipt-Based Budget Tracking: Upload grocery receipts to auto-generate visual spending graphs and edit monthly budgets.
  • Barcode Price Comparison: Scan items to view real-time prices at nearby stores, with map integration for location-aware decisions.
  • AI Meal Suggestions: Personalized recipe recommendations based on pantry contents and dietary preferences.
  • Pantry Management: Visual expiration calendars, barcode-based item entry, and real-time waste reduction alerts.

Technology & Tools

  • Design Tool: Figma (UI/UX design for mobile interface)
  • Core Concepts: Human-Computer Interaction, Usability, Progressive Disclosure, Automation
  • AI Integration: Ingredient-based recipe generation and user prompts for flexible meal planning

Challenges & Future Improvements

  • Enhancing personalization for AI-generated meals
  • Integrating more stores for broader price comparison coverage
  • Adding push notifications for pantry and budgeting reminders
  • Exploring social features like shared meal plans and saving tips

Impact

By applying usability-first design, this app simplifies a high-friction task and promotes smarter, more sustainable choices. It empowers users to shop efficiently, reduce waste, and manage resources with confidence‚Äîall from a single intuitive interface.

Gamified To-Do List Web App

Project Description

This project is a web-based task management tool designed for developers and productivity-focused users who struggle with time estimation, consistency, or motivation. Unlike traditional to-do apps, this tool introduces a gamified system where completing tasks earns virtual currency that can be spent on caring for digital pets. Users who neglect tasks risk losing their pets, reinforcing consistent task tracking through soft behavioral nudges.

The app supports custom task creation, real-time tracking through built-in timers, and visual comparisons between estimated and actual time taken. It encourages time awareness, improves personal planning, and motivates users to stay organized through a combination of visual rewards and progress monitoring. Each task includes features like start/pause controls and completion locking to maintain structure and prevent gaming the system.

Hidden Figures in Physics and Astronomy

Project Overview

The **Hidden Figures in Physics and Astronomy** project is an educational initiative aimed at recognizing and celebrating the contributions of physicists and astronomers from marginalized backgrounds. By providing an accessible, searchable online repository, the platform serves as a valuable resource for educators, students, and researchers to explore the diverse narratives that have shaped these scientific fields.

My Role and Contributions

In this project, I single handedly responsible for the website development, focusing on creating a performant and user-friendly web platform. Leveraging technologies such as **Hugo**, **HTML**, **CSS**, and **JavaScript**, I engineered a scalable architecture that facilitates seamless content management and enhances user engagement. Deigned clear sturctures and lables for the scientists profiles to enable faster index and search for the users.

Key Features and Implementation

  • Static Site Generation with Hugo: Utilized Hugo to build a fast and efficient static site, ensuring quick load times and a smooth user experience.
  • Markdown-Based Content Management: Developed a system where scientist profiles are created and maintained using Markdown files, simplifying content creation and updates.
  • Advanced Search Functionality: Integrated **Pagefind** to enable users to filter profiles by research domain and academic coursework, enhancing discoverability and engagement.
  • Responsive Design: Implemented a responsive layout to ensure accessibility across various devices, improving the platform's reach and usability.

Mission and Impact

The mission of this project is to foster inclusivity and diversity in physics and astronomy education. By highlighting the achievements of underrepresented scientists, the platform aims to support the retention and identity development of marginalized students in these fields. Educators are provided with resources to incorporate diverse role models into their curricula, promoting a more inclusive academic environment.

Technologies Used

  • **Hugo:** Static site generator for building the platform.
  • **HTML/CSS/JavaScript:** Core web technologies for structuring, styling, and adding interactivity to the site.
  • **Pagefind:** Library for implementing advanced search functionalities.
  • **Markdown:** Format for content creation, enabling easy addition and maintenance of scientist profiles.

Repository and Contributions

The project's codebase is hosted on GitHub, allowing for collaborative development and version control. The repository structure includes directories for assets, configurations, content, data, layouts, and static files, facilitating organized and efficient development workflows.

Conclusion

The **Hidden Figures in Physics and Astronomy** platform stands as a testament to the power of technology in promoting diversity and inclusion within the scientific community. Through strategic development and thoughtful design, the project provides a meaningful resource that educates and inspires current and future generations in the fields of physics and astronomy.
